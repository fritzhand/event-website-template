# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do **NOT** create a public GitHub issue.

Instead, please report it via email to the maintainers (insert email here) or use the GitHub Security Advisory feature if available.

We will review the issue and respond as soon as possible.

## Secrets and Sensitive Data

This project is a public template. ensure you:
- Do NOT commit `.env` files.
- Do NOT commit real API keys, tokens, or passwords.
- Use `.env.example` for required environment variables.
