# Contributing to Event Website Template

Thank you for your interest in contributing! We welcome improvements to this template.

## How to Contribute

1. **Fork the repository**
2. **Create a branch** for your feature or fix: `git checkout -b featured/amazing-feature`
3. **Commit your changes**: `git commit -m 'feat: Add amazing feature'`
4. **Push to the branch**: `git push origin feature/amazing-feature`
5. **Open a Pull Request**

## Guidelines

- Keep the template generic and reusable.
- Do not include hardcoded event data; use the `src/content/` pattern.
- Ensure placeholder assets are used for any new visuals.
- Follow the existing code style (Prettier/ESLint).

## Development

Run the development server:

```bash
npm install
npm run dev
```
